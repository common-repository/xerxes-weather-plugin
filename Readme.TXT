== Description ==

Xerxes Weather Plugin is a modifikation of the [Simple Weather](http://wordpress.org/extend/plugins/simple-weather/) Plugin.
The Xerxes Weather Plugin uses the Google API and shows the weather of the current user locaction based on there IP adresse.

The plugin is very easy to use - not settings requierd, you can change the CSS with editor by modifying the weathercss.css

If you got any feedback, bugs or feature request feel free to contact me at [SplusK](http://splusk.de/allgemein/5587-xerxes-weather)

== Installation ==

In most cases you can install automatically from WordPress.

However, if you install this manually, follow these steps:

1. Upload the entire `xerxes-weather` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

FYI: There is no "control panel" for this plugin.
